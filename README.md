# C4STARTERS
A tutorial to use C for complete beginners.

## WHAT IT INCLUDES
It includes these simple things:

*Printing text,
*Compiling,
*Getting basic input,
*Variables,
*If and Else,
*Switch,
*Loops,
*Functions,
*Headers,
*Reading/Writing files,
(And more!)
