THE SYNTAX OF C:

C's syntax is straightforward. For functions, they begin with the function name followed by parentheses. If there are any parameters, they are specified within these parentheses. The function block concludes with a semicolon.
For conditionals, the structure starts with the conditional statement, followed by parentheses containing parameters if applicable. The block of code associated with the conditional is enclosed in curly brackets.
And finally, statements in C start with the statement, and (if there are any), the paremeters will be put after (NOT IN PARENTHESIS) and a semicolon at the end.

Function:

afunction(parameters);

Conditional:

if(1 == 1) {
return 1;
}

Statement:

return 0;

COMMENTS:

The first thing that I am going to do is display a comment. A comment is ignored by the compiler and is essentially an area for notes. you can make a comment by typing "//" and then your text.

// This is a comment. It is ignored by the compiler. You can write notes here

PRINTING TEXT:

Printing text is very simple. To print text, use the "printf()" function

printf("this is the text!\n")

notice that at the end I added "\n" to it. This prints a new line. This is essential to prevent visual bugs. Although there are some scenarios where you can avoid it.

Compiling & running code:

Now, lets try and test the following hello world code that we made. But how do we run it? We use compilers. They get our code and turn it into an executable. There are many options, but I will use gcc as it is very easy to use. to test your code, open your terminal. From there, locate the folder that you put your code into. After that, type this into your terminal:


gcc -o nameofyourprogram nameofthecfile.c

after this, you should be left with an executable. And if you run it, it should print "hello world"
